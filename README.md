# mongo-dashboard
Create a Mongo Dashboard with CubeJs

# How to
1. `./run-container.sh`
2. `docker exec -it dashboard bash`
3. Init a new project `cubejs create <project-name> -d mongobi`
4. `cd <project-name>`
5. `npm run dev`
6.
